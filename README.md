    XAMPP 集成软件

    chrome

## 网站技术摘要

* 兼容当前主流浏览器的展示性公司网站
* 使用`CSS3 media queries`技术，使网站在各个平台都具备良好的浏览体验
* 首页利用`jquery.fullPage.js`搭建，并配合`slick.js`完成切换动效
* 引入了两个有趣的js动效
* 使用地址栏传值，让一个页面能够根据不同的值来加载相对应的内容
